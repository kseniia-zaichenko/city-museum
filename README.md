# CityMuseum

Учебный адаптивный многостраничный сайт городского музея, разработанный в соответствии с индивидуальным заданием по ПМ.05 «Основы верстки сайтов».

## О проекте

Предметная область - городской музей.

Сайт выполнен в стиле старинной книги и музейного архива. В оформлении используются фон состаренной бумаги, книжные шрифты, приглушённые цвета и изображения исторических экспонатов.

Проект состоит из трёх взаимосвязанных страниц:

* главная страница с выставками;
* страница отдельной экспозиции;
* страница с контактами, режимом работы, картой и формой обратной связи.

## Возможности сайта

Пользователь может:

* просматривать действующие и временные выставки;
* открывать подробную страницу экспозиции;
* знакомиться с описанием музейных материалов;
* просматривать краткую историческую хронологию;
* находить адрес и режим работы музея;
* пользоваться схематичной картой;
* заполнять форму обратной связи с HTML-валидацией;
* просматривать сайт на компьютере, планшете и телефоне.

## Запуск

Проект не требует установки дополнительных программ, базы данных или зависимостей.

### Вариант 1. Через Проводник

Открыть файл:

```text
index.html
```

в любом современном браузере.

### Вариант 2. Через PowerShell

Перейти в папку проекта:

```powershell
Set-Location "C:\Users\kseni\Desktop\Синь\пм05\Музей"
```

Открыть главную страницу:

```powershell
Start-Process ".\index.html"
```

### Вариант 3. Через VS Code

1. Открыть папку проекта в Visual Studio Code.
2. Установить расширение Live Server.
3. Нажать правой кнопкой мыши на `index.html`.
4. Выбрать **Open with Live Server**.

## Особенности реализации

* единое оформление всех страниц;
* семантические элементы `header`, `nav`, `main`, `section`, `article` и `footer`;
* один основной заголовок `h1` на каждой странице;
* адаптивные сетки на основе Flexbox и CSS Grid;
* осмысленные альтернативные описания изображений;
* видимый фокус при навигации с клавиатуры;
* связанные с полями формы элементы `label`;
* обязательные поля и проверка формата электронной почты;
* локально сохранённые изображения без зависимости от внешних сервисов.
